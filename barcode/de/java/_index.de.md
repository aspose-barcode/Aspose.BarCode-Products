<p>
 Aspose.BarCode für Java ist eine robuste und zuverlässige API, die Barcode-Erzeugungs- und Erkennungsfunktionen für Java-Anwendungen bereitstellt. Es unterstützt die etabliertesten Standards und Spezifikationen und ermöglicht das Lesen von 1D- und 2D-Barcodes von Bildern unterschiedlicher Formate aus mehreren Winkeln.
</p>

<p>
 Aspose.BarCode für Java rendert die Etiketten in einer Vielzahl von Bildformaten mit hochflexiblen Bildgebungsoptionen. Darüber hinaus bietet die Bibliothek das Drucken sowie das Kodieren und Dekodieren mehrerer Symbologien.
</p>

   <div class="col-lg-12">
    <h2 class="h2title">
     Bearbeiten Sie Etiketten auf viele Arten
    </h2>
    <p>
     Aspose.BarCode for Java API bietet die Möglichkeit, Bildränder, Stil, Rand und Breite sowie die Hintergrund- und Balkenfarbe des Bildes zu manipulieren. Sie können Bilder auch in einen beliebigen Winkel drehen und hochwertige Barcode-Bilder mit Anti-Aliasing erstellen. Darüber hinaus werden Bildränder &amp; Auflösung kann angepasst werden.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Barcodes aus Bildern erkennen
    </h2>
    <p>
     Die Java-Barcode-Bibliothek kann aus verschiedenen Bildern lesen. Die BarCodeReader-Klasse, die hauptsächlich für die Erkennungsfunktion verantwortlich ist, unterstützt viele gängige Bildformate, darunter BMP, PNG, TIFF, JPEG &amp; GIF. Der Erkennungsprozess ist hochgradig konfigurierbar, sodass Sie erwartete Regionen festlegen können, um Codeetiketten auf einem Bild zu finden, manuelle Erkennungshinweise anzuwenden, um den Scan &amp; Wenden Sie Binarisierungsfilter auf das Bild an.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Barcode-Typ und Text aus Bild lesen - Java
     </h3>
     <pre><code class="java">// Lesegerät mit Bild initialisieren &amp; erwarteter Typ

BarCodeReader reader = new BarCodeReader(dir + "bcode.png", DecodeType.ALL_SUPPORTED_TYPES);

// Code lesen

while (reader.read()) {

	// Codetyp und Text drucken

	System.out.println("Type: " + reader.getCodeType() + " Text: " + reader.getCodeText());

}</code></pre>
