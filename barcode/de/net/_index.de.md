<p>Aspose.BarCode für .NET ist eine leistungsstarke Bibliothek zum Generieren und Erkennen von 1D- und 2D-Barcodes aus mehreren Bildtypen in jedem Winkel. Entwickler können ihre .NET-Anwendungen problemlos um die Barcode-Erzeugungs- und Scanfunktionalität erweitern. Es ermöglicht auch den Export der generierten Barcodes in verschiedene hochwertige Bildformate.</p>

<p>Aspose.BarCode für .NET unterstützt die Manipulation des Aussehens der generierten Barcodes wie Hintergrundfarbe, Balkenfarbe, Rotationswinkel, x-Dimension, Bildqualität, Auflösung, Beschriftungen, Größe und vieles mehr. Darüber hinaus unterstützt die API mehrere Arten von Erkennungs- und Bildfunktionen sowie verschiedene Arten von Symbologien.</p>

    <h2 class="h2title">
     Barcodes generieren
    </h2>
    <p>
     Die Barcode-Generator-API bietet die Möglichkeit, die Barcodes zu erstellen, indem verschiedene Symbologien angegeben, Codetext (Daten, die in das Barcodebild codiert werden sollen) und erscheinungsbezogene Eigenschaften wie Hintergrundfarbe, Schriftart sowie Position (verbergen, oben, unten) und Ausrichtung festgelegt werden . Darüber hinaus kann man zweidimensionale Barcodes wie PDF 417, Data Matrix, Aztec, QR und Micro QR Barcode usw. auf die gleiche Weise wie eindimensionale erstellen.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Barcode-Generierung – C#
     </h3>
     <pre><code class="cs"> // Der Pfad zum Dokumentenverzeichnis.

string dataDir = "define-your-directory";

// Objekt instanziieren und verschiedene Barcode-Eigenschaften festlegen

BarcodeGenerator  generator = new BarcodeGenerator (EncodeTypen.Code128, "1234567");

generator.Parameters.Barcode.XDimension.Millimeters = 1f;



// Speichern Sie das Bild auf Ihrem System und stellen Sie das Bildformat auf JPEG ein

generator.Save(dataDir + "output.jpg", BarCodeImageFormat.Jpeg);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Unterstützung für Bildgebung, Rotation und Größenanpassung
    </h2>
    <p>
     Aspose.BarCode für .NET ermöglicht die Bearbeitung von Etikettenrahmen, Rahmenfarben, Stil, Rändern und Breite. Es bietet auch die Möglichkeit, Bildfarbe, Hintergrundfarbe und Balkenfarbe zu ändern. Sie können Barcode-Bilder auch in einen beliebigen Winkel drehen und hochwertige Bilder mit Anti-Aliasing erstellen. Darüber hinaus können Sie benutzerdefinierte Auflösungen für die resultierenden Bilder festlegen, während Sie sie in einem der unterstützten Bildformate speichern.
    </p>
