<p>
 Aspose.BarCode für PHP über Java ist eine Reihe von APIs zum Generieren und Lesen von Barcodes für PHP-Anwendungen und -Skripte. Entwickler können problemlos 1D-, 2D- und Post-Barcodes aus Bildern aus verschiedenen Winkeln lesen. Die Barcode-Generator-API erstellt Barcode-Bilder in verschiedenen Formaten wie JPG, PNG, GIF, BMP. Darüber hinaus unterstützt es mehrere Bildgebung-Funktionen wie das Bearbeiten von Rändern, Farbe, Drehen von Bildern, Anpassen der Auflösung und vieles mehr.
</p>
