<p>
 Aspose.BarCode för Node.js via Java är ett lätt anpassningsbart API för streckkodsskanning och generering. Utvecklare kan integrera det via nodejs-java bridge. Det är en snabb och enkel streckkodsavkodare API för att avkoda vanliga 1D, 2D streckkoder var som helst i valfri vinkel från en bild. Man kan enkelt ange ett område i bilden för att skanna streckkoden. API genererar streckkodsbilder i flera format som JPG, PNG, GIF, BMP samt stöder flera bildbehandlingsfunktioner som manipulering av gränser och stilar, färg, marginaler, roterande bilder, anpassa upplösning och mycket mer.
</p>
