<p>
 Aspose.BarCode för Android via Java är en uppsättning streckkodsgenerering &amp; skanna API:er för Android-applikationer. API stöder 1D, 2D &amp; poststreckkoder från olika vinklar på bilder. Streckkodsgenerator API skapar enkelt streckkodsbilder i flera format. Dessutom stöder den flera bildbehandlingsfunktioner som att manipulera bildkanter, bildfärg, rotera bilder, anpassa upplösning och mycket mer.
</p>
