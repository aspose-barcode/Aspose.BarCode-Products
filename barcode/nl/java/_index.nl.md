<p>
 Aspose.BarCode voor Java is een robuuste en betrouwbare API die voorziet in het genereren en herkennen van streepjescodes voor Java-toepassingen. Het ondersteunt de meest gevestigde normen en specificaties en maakt het mogelijk om 1D- en 2D-barcodes van afbeeldingen in verschillende formaten vanuit meerdere hoeken te lezen.
</p>

<p>
 Aspose.BarCode voor Java geeft de labels weer in een groot aantal afbeeldingsformaten met zeer flexibele beeldopties. Bovendien biedt de bibliotheek het printen en coderen en decoderen van meerdere symbolieken.
</p>

    <h2 class="h2title">
     Labels op vele manieren manipuleren
    </h2>
    <p>
     Aspose.BarCode voor Java API biedt de mogelijkheid om randen, stijl, marge en breedte van afbeeldingen te manipuleren, evenals de achtergrondkleur en balkkleur van de afbeelding. U kunt afbeeldingen ook naar elke hoek draaien en hoogwaardige barcodeafbeeldingen produceren met anti-aliasing. Daarnaast zijn afbeeldingsmarges &amp; resolutie kan worden aangepast.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Herken streepjescodes van afbeeldingen
    </h2>
    <p>
     Java-barcodebibliotheek kan uit verschillende afbeeldingen lezen. BarCodeReader-klasse, die voornamelijk verantwoordelijk is voor de herkenningsfunctie, ondersteunt veel populaire afbeeldingsindelingen, waaronder BMP, PNG, TIFF, JPEG &amp; GIF. Het herkenningsproces is zeer configureerbaar, zodat u verwachte regio's kunt instellen om codelabels op een afbeelding te vinden, handmatige herkenningstips kunt toepassen om de scan te optimaliseren &amp; binarisatiefilters toepassen op de afbeelding.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Lees barcodetype en tekst uit afbeelding - Java

BarCodeReader reader = new BarCodeReader(dir + "bcode.png", DecodeType.ALL_SUPPORTED_TYPES);

// lees code

while (reader.read()) {

	// print het codetype en de tekst
