<p>
 Aspose.BarCode voor Android via Java is een set voor het genereren van barcodes &amp; het scannen van API's voor Android-applicaties. API ondersteunt 1D, 2D &amp; post barcodes vanuit verschillende hoeken op afbeeldingen. Barcode generator API maakt eenvoudig barcode-afbeeldingen in meerdere formaten. Bovendien ondersteunt het meerdere beeldfuncties, zoals het manipuleren van afbeeldingsranden, afbeeldingskleuren, het roteren van afbeeldingen, het aanpassen van de resolutie en nog veel meer.
</p>
