<p>
 Το Aspose.BarCode για το Node.js μέσω Java είναι ένα εύκολα προσαρμόσιμο API σάρωσης και δημιουργίας γραμμικού κώδικα. Οι προγραμματιστές μπορούν να το ενσωματώσουν μέσω nodejs-java bridge. Είναι ένα γρήγορο και απλό API αποκωδικοποιητή γραμμωτού κώδικα για την αποκωδικοποίηση κοινών γραμμωτών κωδίκων 1D, 2D οπουδήποτε και σε οποιαδήποτε γωνία από μια εικόνα. Μπορεί κανείς εύκολα να καθορίσει μια περιοχή στην εικόνα για να σαρώσει τον γραμμωτό κώδικα. Το API δημιουργεί εικόνες γραμμωτού κώδικα σε πολλαπλές μορφές, όπως JPG, PNG, GIF, BMP, καθώς και υποστηρίζει πολλαπλές δυνατότητες απεικόνισης, όπως χειρισμό περιγραμμάτων και στυλ, χρώμα, περιθώρια, περιστρεφόμενες εικόνες, προσαρμογή ανάλυσης και πολλά άλλα.
