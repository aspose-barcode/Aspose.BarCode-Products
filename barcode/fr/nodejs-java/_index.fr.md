<p>
 Aspose.BarCode pour Node.js via Java est une API de numérisation et de génération de codes-barres facilement personnalisable. Les développeurs peuvent l'intégrer via le pont nodejs-java. Il s'agit d'une API de décodeur de codes-barres simple et rapide pour décoder les codes-barres 1D et 2D courants n'importe où, sous n'importe quel angle à partir d'une image. On peut facilement spécifier une zone dans l'image pour scanner le code-barres. L'API génère des images de codes-barres dans plusieurs formats tels que JPG, PNG, GIF, BMP et prend en charge plusieurs fonctionnalités d'imagerie telles que la manipulation des bordures et des styles, la couleur, les marges, la rotation des images, la personnalisation de la résolution et bien plus encore.
</p>
