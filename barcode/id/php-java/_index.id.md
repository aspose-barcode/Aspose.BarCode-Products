<p>
 Aspose.BarCode untuk PHP melalui Java adalah satu set pembuatan kode batang dan membaca API untuk aplikasi dan skrip PHP. Pengembang dapat dengan mudah membaca kode batang 1D, 2D, dan pos dari gambar pada sudut yang berbeda. API generator kode batang membuat gambar kode batang dalam berbagai format seperti JPG, PNG, GIF, BMP. Selain itu, Ini mendukung beberapa fitur pencitraan seperti memanipulasi batas, warna, memutar gambar, menyesuaikan resolusi dan banyak lagi.
