<p>
 Aspose.BarCode untuk Node.js melalui Java adalah pemindaian kode batang dan API generasi yang mudah disesuaikan. Pengembang dapat mengintegrasikannya melalui jembatan nodejs-java. Ini adalah API dekoder kode batang yang cepat dan sederhana untuk memecahkan kode kode batang 1D, 2D umum di mana saja dan dari sudut mana pun dari gambar. Seseorang dapat dengan mudah menentukan area dalam gambar untuk memindai kode batang. API menghasilkan gambar barcode dalam berbagai format seperti JPG, PNG, GIF, BMP serta mendukung beberapa fitur pencitraan seperti memanipulasi batas dan gaya, warna, margin, memutar gambar, menyesuaikan resolusi, dan banyak lagi.
</p>
