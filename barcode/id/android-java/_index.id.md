<p>
 Aspose.BarCode untuk Android melalui Java adalah satu set pembuatan kode batang &amp; memindai API untuk aplikasi Android. API mendukung 1D, 2D &amp; barcode pos dari sudut yang berbeda pada gambar. API generator kode batang dengan mudah membuat gambar kode batang dalam berbagai format. Selain itu, Ini mendukung beberapa fitur pencitraan seperti memanipulasi batas gambar, warna gambar, memutar gambar, menyesuaikan resolusi dan banyak lagi.
</p>
