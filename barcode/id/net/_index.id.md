<p>Aspose.BarCode untuk .NET adalah perpustakaan yang kuat untuk menghasilkan dan mengenali barcode 1D dan 2D dari beberapa jenis gambar di setiap sudut. Pengembang dapat dengan mudah menambahkan pembuatan kode batang dan fungsionalitas pemindaian ke aplikasi .NET mereka. Ini juga memungkinkan mengekspor kode batang yang dihasilkan ke berbagai format gambar berkualitas tinggi.</p>

<p>Aspose.BarCode untuk .NET mendukung untuk memanipulasi tampilan kode batang yang dihasilkan seperti warna latar belakang, warna batang, sudut rotasi, dimensi-x, kualitas gambar, resolusi, keterangan, ukuran, dan banyak lagi. Selain itu, API mendukung berbagai jenis fitur pengenalan dan pencitraan serta berbagai jenis simbologi.</p>

    <h2 class="h2title">
     Hasilkan Barcode
    </h2>
    <p>
     API generator kode batang menawarkan untuk membuat kode batang dengan menentukan berbagai simbol, mengatur teks kode (data yang akan dikodekan ke gambar kode batang) dan properti terkait tampilan seperti warna latar depan, font serta lokasi (sembunyikan, di atas, di bawah) dan perataan . Selain itu, seseorang dapat membuat barcode dua dimensi seperti PDF 417, Data Matrix, Aztec, QR dan Micro QR barcode dll dengan cara yang sama seperti satu dimensi.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Pembuatan Kode Batang - C#
     </h3>
     <pre><code class="cs"> // Jalur ke direktori dokumen.

string dataDir = "define-your-directory";

// Buat instance objek dan atur properti barcode yang berbeda

BarcodeGenerator  generator = new BarcodeGenerator (EncodeJenis.Code128, "1234567");

generator.Parameters.Barcode.XDimension.Millimeters = 1f;



// Simpan gambar ke sistem Anda dan atur format gambarnya ke Jpeg

generator.Save(dataDir + "output.jpg", BarCodeImageFormat.Jpeg);</code></pre>
