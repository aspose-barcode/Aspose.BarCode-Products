<p>
 Aspose.BarCode for Node.js via Java 是一個易於定制的條形碼掃描和生成 API。開發者可以通過 nodejs-java 橋接器來集成它。它是一種快速簡單的條形碼解碼器 API，可以從圖像的任何角度對常見的一維、二維條碼進行解碼。人們可以很容易地在圖像中指定一個區域來掃描條形碼。 API 生成多種格式的條形碼圖像，如 JPG、PNG、GIF、BMP，並支持多種成像功能，如操作邊框和样式、顏色、邊距、旋轉圖像、自定義分辨率等等。
