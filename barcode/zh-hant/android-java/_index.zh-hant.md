<p>
 Aspose.BarCode for Android via Java 是一套條碼生成&amp;掃描 Android 應用程序的 API。 API 支持 1D、2D 和圖像上不同角度的郵政條形碼。條碼生成器 API 可輕鬆創建多種格式的條碼圖像。此外，它還支持多種成像功能，例如操作圖像邊框、圖像顏色、旋轉圖像、自定義分辨率等等。
</p>
