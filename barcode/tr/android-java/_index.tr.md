<p>
 Aspose.BarCode for Android, Java aracılığıyla bir dizi barkod oluşturma &amp; Android uygulamaları için tarama API'leri. API, 1D, 2D &amp; resimlerde farklı açılardan posta barkodları. Barkod oluşturucu API, birden çok formatta kolayca barkod görüntüleri oluşturur. Ayrıca, görüntü kenarlıklarını değiştirme, görüntü rengi, görüntüleri döndürme, çözünürlüğü özelleştirme ve çok daha fazlası gibi çoklu görüntüleme özelliklerini destekler.
</p>
