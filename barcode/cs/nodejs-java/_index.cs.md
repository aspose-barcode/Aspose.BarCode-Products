<p>
 Aspose.BarCode for Node.js přes Java je snadno přizpůsobitelné rozhraní API pro skenování a generování čárových kódů. Vývojáři jej mohou integrovat přes nodejs-java bridge. Jedná se o rychlé a jednoduché rozhraní API dekodéru čárových kódů pro dekódování běžných 1D, 2D čárových kódů kdekoli v libovolném úhlu z obrázku. Je možné snadno určit oblast na obrázku pro skenování čárového kódu. API generuje obrázky čárových kódů v různých formátech, jako jsou JPG, PNG, GIF, BMP, a také podporuje různé zobrazovací funkce, jako je manipulace s okraji a styly, barva, okraje, otáčení obrázků, přizpůsobení rozlišení a mnoho dalšího.
</p>
