<p>
 Aspose.BarCode pro Android přes Java je sada pro generování čárových kódů &amp; rozhraní API pro skenování pro aplikace Android. API podporuje 1D, 2D &amp; poštovní čárové kódy z různých úhlů na obrázcích. Generátor čárových kódů API snadno vytváří obrázky čárových kódů v různých formátech. Navíc podporuje více zobrazovacích funkcí, jako je manipulace s okraji obrazu, barva obrazu, otáčení obrázků, přizpůsobení rozlišení a mnoho dalšího.
</p>
