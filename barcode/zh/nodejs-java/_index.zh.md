<p>
 Aspose.BarCode for Node.js via Java 是一个易于定制的条形码扫描和生成 API。开发者可以通过 nodejs-java 桥接器来集成它。它是一种快速简单的条形码解码器 API，可以从图像的任何角度对常见的一维、二维条码进行解码。人们可以很容易地在图像中指定一个区域来扫描条形码。 API 生成多种格式的条形码图像，如 JPG、PNG、GIF、BMP，并支持多种成像功能，如操作边框和样式、颜色、边距、旋转图像、自定义分辨率等等。
