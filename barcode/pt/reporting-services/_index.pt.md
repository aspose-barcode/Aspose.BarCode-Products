<p>
 Aspose.BarCode for Reporting Services é uma solução abrangente para renderização de códigos de barras lineares e 2D no Microsoft SQL Server Reporting Services. As etiquetas de código de barras podem ser codificadas com caracteres não ingleses e renderizadas nos formatos BMP, JPEG, GIF e PNG.
