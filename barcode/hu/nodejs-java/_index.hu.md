<p>
 Az Aspose.BarCode for Node.js a Java-n keresztül egy könnyen testreszabható vonalkód-leolvasó és -generáló API. A fejlesztők a nodejs-java hídon keresztül integrálhatják. Ez egy gyors és egyszerű vonalkód-dekódoló API, amely az általános 1D, 2D vonalkódokat bárhol, bármilyen szögben dekódolja a képről. Könnyen megadható egy terület a képen a vonalkód beolvasásához. Az API többféle formátumban, például JPG, PNG, GIF, BMP vonalkódképeket generál, valamint több képalkotási funkciót is támogat, mint például a szegélyek és stílusok, színek, margók manipulálása, képek elforgatása, felbontás testreszabása és még sok más.
</p>
