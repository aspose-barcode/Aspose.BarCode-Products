<p>
 Aspose.BarCode para Python a través de Java es un conjunto de API de generación y lectura de códigos de barras para aplicaciones de Python mediante el uso del puente python-java. Los desarrolladores pueden agregar fácilmente la funcionalidad de generación y reconocimiento de códigos de barras. La API de código de barras de Python crea imágenes de código de barras en varios formatos, como PNG, JPG, GIF, BMP. Además, es compatible con múltiples funciones de imágenes, como la manipulación de bordes, el color, la rotación de imágenes, la personalización de la resolución y mucho más.
</p>
