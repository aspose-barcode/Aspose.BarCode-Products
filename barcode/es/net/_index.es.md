<p>Aspose.BarCode para .NET es una poderosa biblioteca para generar y reconocer códigos de barras 1D y 2D a partir de múltiples tipos de imágenes en cualquier ángulo. Los desarrolladores pueden agregar fácilmente la funcionalidad de generación y escaneo de códigos de barras a sus aplicaciones .NET. También permite exportar los códigos de barras generados a diferentes formatos de imagen de alta calidad.</p>

<p>Aspose.BarCode para .NET admite la manipulación de la apariencia de los códigos de barras generados, como el color de fondo, el color de la barra, el ángulo de rotación, la dimensión x, la calidad de la imagen, la resolución, los subtítulos, el tamaño y mucho más. Además, la API admite múltiples tipos de funciones de reconocimiento e imágenes, así como diferentes tipos de simbologías.</p>

    <h2 class="h2title">
     Generar códigos de barras
    </h2>
    <p>
     La API del generador de código de barras ofrece la creación de códigos de barras especificando varias simbologías, configurando el texto del código (datos que se codificarán en la imagen del código de barras) y propiedades relacionadas con la apariencia, como el color de primer plano de fondo, la fuente y la ubicación (ocultar, arriba, abajo) y alineación . Además, se pueden crear códigos de barras bidimensionales como PDF 417, Data Matrix, Aztec, QR y Micro QR, etc. de la misma manera que unidimensional.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Generación de código de barras - C#
     </h3>
     <pre><code class="cs"> // La ruta al directorio de documentos.

string dataDir = "define-your-directory";

// Crea una instancia del objeto y establece las diferentes propiedades del código de barras

BarcodeGenerator  generator = new BarcodeGenerator (EncodeTipos.Code128, "1234567");

generator.Parameters.Barcode.XDimension.Millimeters = 1f;



// Guarde la imagen en su sistema y establezca su formato de imagen en Jpeg

generator.Save(dataDir + "output.jpg", BarCodeImageFormat.Jpeg);</code></pre>
    </div>

También ofrece la posibilidad de modificar el color de la imagen, el color de fondo y el color de la barra. También puede rotar imágenes de códigos de barras en cualquier ángulo y producir imágenes de alta calidad con suavizado. Además de esto, puede establecer resoluciones personalizadas para las imágenes resultantes mientras las guarda en cualquiera de los formatos de imagen admitidos.
