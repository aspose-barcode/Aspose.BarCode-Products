<p>
 Aspose.BarCode para Node.js a través de Java es una API de generación y escaneo de códigos de barras fácilmente personalizable. Los desarrolladores pueden integrarlo a través del puente nodejs-java. Es una API de decodificador de código de barras rápida y simple para decodificar códigos de barras 1D y 2D comunes en cualquier lugar y en cualquier ángulo de una imagen. Uno puede especificar fácilmente un área en la imagen para escanear el código de barras. La API genera imágenes de códigos de barras en múltiples formatos, como JPG, PNG, GIF, BMP, y admite múltiples funciones de imágenes, como la manipulación de bordes y estilos, color, márgenes, rotación de imágenes, resolución personalizada y mucho más.
</p>
