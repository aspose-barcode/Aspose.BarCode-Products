<p>
 Aspose.BarCode for Reporting Services es una solución integral para representar códigos de barras lineales y 2D dentro de Microsoft Servicios de informes de SQL Server. Las etiquetas de código de barras se pueden codificar con caracteres no ingleses y se pueden representar en formatos BMP, JPEG, GIF y PNG.
</p>
