<p>
 Aspose.BarCode para PHP a través de Java es un conjunto de API de generación y lectura de códigos de barras para aplicaciones y scripts de PHP. Los desarrolladores pueden leer fácilmente códigos de barras 1D, 2D y postales de imágenes en diferentes ángulos. La API del generador de códigos de barras crea imágenes de códigos de barras en varios formatos, como JPG, PNG, GIF, BMP. Además, es compatible con múltiples funciones de imágenes, como la manipulación de bordes, el color, la rotación de imágenes, la personalización de la resolución y mucho más.
</p>
